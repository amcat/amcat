If-syntax

The if-column (`condition`) of a coding rule determines if the action associated with it be performed. If it does match, the target field can be painted red, marked not null or non-codable depending on the value of the apply-column.

To provide a powerful way of expressing conditions, a user can denote his condition in a programming-like manner. The syntax used is a subset of the syntax of Python programming language.
